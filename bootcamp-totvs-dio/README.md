# 🚀 Bootcamp TOTVS – Fundamentos de Engenharia de Dados e Machine Learning

> Repositório com todos os projetos desenvolvidos durante o **Bootcamp TOTVS** na plataforma **DIO (Digital Innovation One)**.

---

## 🎯 Sobre o Bootcamp

O Bootcamp TOTVS é um programa intensivo que combina **Engenharia de Dados**, **Machine Learning** e **Computação em Nuvem**, unindo ferramentas tradicionais como Python, SQL e Excel ao poder da IA Generativa e da AWS.

**Parceria:** TOTVS + Digital Innovation One (DIO)
